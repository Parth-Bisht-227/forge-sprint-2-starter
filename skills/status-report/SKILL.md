# Status Report Skill

## Description
Generates a comprehensive status report of the project's current progress against the `TASKS.md` checklist and `agent-log.md`.

## Procedure
1. Read `TASKS.md` to identify completed and pending items.
2. Parse `agent-log.md` for the most recent significant events and blockers.
3. Compare current state against the `PROJECT.md` goals.
4. Format the report with:
   - **Executive Summary**: High-level status (On Track/At Risk/Blocked).
   - **Completed Work**: List of recently finished tasks.
   - **Upcoming Milestones**: Next 3 critical tasks.
   - **Blockers**: Any identified issues requiring human intervention.
5. Output the report in a concise, professional format.
